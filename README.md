## Game Controls
     *Change music = M
### Controls Player 1
     * Move Car = WASD
     * Nitro = SPACE
### Controls Player 2
     * Move Car = ↑ ← ↓ →
     * Nitro = SHIFT
### Debug Mode
     * F1 = Shows/hides collisions
     * F2 = Add one lap to Player 1
     * F3 = Add one lap to Player 2
     * F4 = Recover boost quantity to Player 1
     * F5 = Recover boost quantity to Player 2
